# BitCrawler

2D 픽셀아트 로그라이크 게임 — Python + Pygame-CE

---

## 실행 방법

```bash
pip install pygame-ce
python3 main.py
```

---

## 조작키

| 키 | 기능 |
|----|------|
| `W A S D` / 방향키 | 이동 (꾹 누름으로 연속 이동) |
| 이동 → 적 위치 | 근접 공격 |
| `1` ~ `5` | 인벤토리 슬롯 아이템 사용 |
| `Q` | 스킬: 돌진 (3칸 전진, 쿨다운 3초) |
| `E` | 스킬: 회오리 (8방향 전체 공격, 쿨다운 5초) |
| `F` | 스킬: 치유 (HP 30% 회복, 쿨다운 8초) |
| `.` / `Space` | 대기 |
| `R` (사망 후) | 재시작 |
| `ESC` | 종료 (상점 내에서는 상점 닫기) |
| `L` (메뉴) | 이어하기 |

## 구현된 시스템

### 화면 레이아웃 (1024 × 768)

```
┌──────────────────────────────┬──────────────┐
│         Floor N   Lv.N       │              │
├──────────────────────────────┤   HP / XP    │
│                              │   ATK / DEF  │
│                              ├──────────────┤
│        게임 뷰포트            │  인벤토리    │
│      (25 × 19 타일)          │  [1] ...     │
│                              │  [2] ...     │
│                              ├──────────────┤
├──────────────────────────────┤  조작키 안내 │
│  메시지 로그                 │              │
└──────────────────────────────┴──────────────┘
```

### 맵 생성

- 80 × 60 타일 그리드, 방 최대 18개
- 랜덤 방 배치 → L자 복도 연결
- **안개(Fog of War)** : Bresenham 시야선, 반경 7타일
  - 밝음(현재 시야) / 어두움(탐험함) / 검정(미탐험) 3단계

### 플레이어

| 항목 | 내용 |
|------|------|
| 초기 스탯 | HP 30, ATK 5, DEF 2 |
| 레벨업 | XP 획득 시 자동 → HP / ATK / DEF 상승 |
| 인벤토리 | 최대 5칸 |
| 이동 | WASD / 방향키, 꾹 누름 연속 이동 |

### 전투

- **범프 전투** : 적 위치로 이동하면 자동 공격
- **데미지 공식** : `max(1, ATK - DEF + rand(0~3))`
- **치명타** : 10% 확률, 2배 데미지
- **계단** : 밟으면 다음 층으로 이동 (무한 진행)

### 실시간 적 AI

적은 플레이어 행동과 **무관하게 독립 타이머로 행동**합니다.

| 몬스터 | 이동 주기 | 공격 주기 |
|--------|---------|---------|
| 쥐 | 0.55 초 | 1.1 초 |
| 고블린 | 0.82 초 | 1.4 초 |
| 스켈레톤 | 0.98 초 | 1.3 초 |
| 오크 | 1.1 초 | 1.7 초 |
| 트롤 | 1.4 초 | 2.1 초 |

- 인식 범위 14타일, 추적 범위 10타일
- 스폰 시 타이머 랜덤 오프셋 → 동시 공격 방지

### 아이템 (JSON 데이터 주도)

| 종류 | 아이템 |
|------|--------|
| 소비 | 체력 물약 (HP +20), 대형 체력 물약 (HP +45) |
| 무기 | 단검 (ATK +2), 장검 (ATK +4) |
| 방어구 | 가죽 갑옷 (DEF +1), 사슬 갑옷 (+2), 플레이트 (+3) |

### 스프라이트 (절차적 픽셀아트)

외부 이미지 파일 없이 `pygame.draw` 로 모든 그래픽을 생성합니다.

**플레이어** — 4방향 × 2프레임 걷기 모션

| 방향 | 특징 |
|------|------|
| 남(↓) | 정면 — 바이저, 방패, 검 |
| 북(↑) | 후면 — 투구 뒷통수, 등판 |
| 서(←) | 좌측 프로파일 — 검을 앞으로 |
| 동(→) | 우측 프로파일 (좌측 반전) |

**몬스터 5종** — 쥐 / 고블린 / 스켈레톤 / 오크 / 트롤

### 애니메이션

| 이펙트 | 내용 | 지속 시간 |
|--------|------|---------|
| 돌진 (Lunge) | 공격 방향으로 튀어나갔다 복귀 | 240 ms |
| 슬래시 (Slash) | 팬 형태 5선 + 충격 링 | 280 ms |
| 피격 플래시 | 타일 번쩍 + 데미지 숫자 부유 | 480 ms |
| 치명타 | 슬래시 색상 빨간색으로 변경 | 280 ms |

### UI

- **한국어 폰트** : AppleSDGothicNeo 자동 감지 (Windows / Linux 폴백 포함)
- **IME 우회** : `pygame.key.get_pressed()` 기반 — 한국어 입력 모드에서도 정상 동작
- **메시지 로그** : 흰색(정보) / 노랑(공격·경고) / 초록(획득·레벨업) / 빨강(피해)

## Phase 3 추가 기능

### 미니맵
- 우측 패널 하단에 2px/타일 규모로 전체 맵(80×60) 표시
- 탐험한 타일(어둡게) / 현재 시야(밝게) / 플레이어(흰점) / 적(빨강) / 보스(보라) / 계단(노랑) / 상점(초록)

### 골드 + 상점
- 적 처치 시 골드 드롭 (쥐 2G ~ 트롤 30G)
- **3의 배수 층**에 상점 등장 — 상점 타일을 밟으면 구매 UI 표시
- 키 `1`~`4`로 아이템 구매, `ESC`로 나가기

### 스킬 시스템
| 키 | 스킬 | 효과 | 쿨다운 |
|----|------|------|--------|
| `Q` | 돌진 | 3칸 전진(적 충돌 시 공격) | 3초 |
| `E` | 회오리 | 8방향 동시 공격 | 5초 |
| `F` | 치유 | HP 30% 회복 | 8초 |

### 보스 몬스터
- **5의 배수 층**에 보스 등장 (계단 없음 → 보스 처치 후 계단 출현)
- 보스 HP 바가 게임 뷰 상단에 표시됨
- 어둠의 기사 (HP 130, ATK 18) / 리치 (HP 110, ATK 24) 교대 등장
- 처치 시 150~200 G 드롭

### 사운드
- 공격 / 피격 / 아이템 획득·사용 / 레벨업 / 계단 / 스킬 / 보스 등장 등
- 외부 파일 없이 `math.sin + struct.pack`으로 8비트 절차적 생성

### 저장/불러오기
- 층 이동 시 자동 저장 (`savegame.json`)
- 사망 시 세이브 삭제
- 타이틀 메뉴 `[ Space ] 새 게임` / `[ L ] 이어하기`
